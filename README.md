編譯環境：Eclipse
JDK版本：jre1.7.0_21
Android SDK版本：2.2.2
Facebook SDK版本：3.0.2

各個Activity介紹：

Gps_project:
1.顯示Google Maps
2.登入Facebook、顯示用戶名稱
3.取得及上傳GPS資料
4.取得Server的軌跡資訊

Places:
1.停留點附近的景點清單

CheckInPage:
1.從系統相簿選取該停留點時間附近的圖片
2.增加訊息
3.標記好友
4.發布更新

PickFriendsActivity:
1.開啟選取好友頁面

FriendsPickerApplication
1.儲存被選取的好友
